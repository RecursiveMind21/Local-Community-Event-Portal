# 🏘️ Local Community Event Portal

## 📋 Project Overview
A lightweight, browser-based community event portal developed for the Local City Council. Enables residents to register for events, view event galleries, access location-based services, and save preferences.

## 🎯 Features Implemented
- ✅ Semantic HTML5 with cross-browser compatibility
- ✅ Responsive navigation with internal and external linking
- ✅ Event registration form with real-time validation
- ✅ Image gallery with double-click magnification
- ✅ Video integration with media event handling
- ✅ LocalStorage & SessionStorage for user preferences
- ✅ Geolocation for nearby event discovery
- ✅ Comprehensive form event handling (blur, change, click, keyup)
- ✅ Mobile-responsive design

## 🛠️ Technologies Used
- HTML5 (Semantic Elements)
- CSS3 (Flexbox, Grid, Animations)
- Vanilla JavaScript (ES6+)
- Chrome DevTools for debugging
- LocalStorage / SessionStorage APIs
- Geolocation API
