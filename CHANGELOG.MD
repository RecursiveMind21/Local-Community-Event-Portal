# Changelog

## [4.0.0] - 2026-06-01
### Added
- Professional multi-page architecture (separate HTML files)
- Shared CSS (`style.css`) and JavaScript (`main.js`) modules
- External help documentation page (`help.html`)
- Improved responsive design for mobile devices

### Changed
- Restructured from single-page to 5-page application
- Cleaner navigation with active state highlighting

## [3.0.0] - 2026-05-30
### Added
- Event registration form with full validation
- localStorage preference saving (event type persists)
- Geolocation for nearby events (with permission & timeout handling)
- Video element with `oncanplay` event
- `onbeforeunload` warning for unsaved form changes

### Fixed
- Phone number validation on blur
- Character counter on feedback textarea
- Double-click image enlargement

## [2.0.0] - 2026-05-29
### Added
- Registration form structure (name, email, date, event type, message)
- Form event handlers (blur, change, click, keyup)
- Basic form styling

### Changed
- Improved navigation links

## [1.0.0] - 2026-05-27
### Added
- Base HTML5 template with `<!DOCTYPE html>`, `lang="en"`, `meta charset`
- Navigation bar with anchor links to sections
- Image gallery using `<table>` (2 rows × 3 columns)
- Welcome banner with inline styles and `.highlight` class
- Footer with external help link

## [0.1.0] - 2026-05-25
### Added
- Initial project setup (empty repository)
